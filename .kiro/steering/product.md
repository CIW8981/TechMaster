# Product Overview

TechMaster Learning Notes is a comprehensive documentation platform for technology learning and AWS certification preparation. Built with MkDocs Material, it provides searchable, well-organized technical documentation covering AWS services, certifications, and related technologies like Docker, Kubernetes, and Terraform.

## Purpose

- Centralized knowledge base for AWS technologies organized by service categories (compute, storage, networking, databases, security, management)
- Structured certification study materials for AWS Cloud Practitioner, Solutions Architect, Developer, and SysOps exams
- Documentation for complementary technologies (Docker, Kubernetes, Terraform, programming)
- Shared resources including glossaries, cheat sheets, and practice labs

## Key Features

- Full-text search across all documentation
- Syntax-highlighted code examples with copy functionality
- Mermaid diagrams for architecture visualization
- Dark/light theme toggle
- Mobile-responsive design
- Git revision dates showing content freshness
- Tag-based content organization
- Cross-referenced topics with internal linking

## Target Audience

Developers and engineers studying for AWS certifications or learning cloud technologies, DevOps tools, and infrastructure as code.
